    Tags that I have learned:
  • b - Bold Text
  • strong - Like Bold Text, but adds importance for HTML-readers
  • i - Italic text
  • em - Similar to strong, but for italicized text; Used for emphasis; Can cause readers to read in a different accent
  • mark - Highlighted text
  • small - Small text
  • del - Strikethrough text
  • u - Underlined text
  • sub - Subscript
  • sup - Superscript

  • ul - Unordered List
  • ol - Ordered List
  • li - List Item
  • img - Image
  • blockquote - Quote Block
  • q - Quote
  • abbr - Abbreviation
  • address - Contact information for an article's source/author
  • cite - Used to cite creative works
  • bdo - Bi-Directional Overide --> Used writing from LTR to RTL
    Attributes I learned:
  1. text-align - Left, Right, Center; Aligns text horizontally within the element
  2. background-color - Changes the background color of the element
  3. color - Changes the font color of the element
  4. title - Provides information to screen readers and
  5. font-family - Changes the font used in an element
  6. font-size - Changes the default font size of an element

I am about to learn CSS and the three different ways it can be used in an HTML file

  • Inline - Using the "style" attribute; good for when only one style needs to be changed
  • Internal - Using a "style" element in the "head" section
  • External - Creating a separate file that can be accessed using the "link" tag in the "head" section (Most common)
